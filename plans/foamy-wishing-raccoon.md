# NotebookLM 調査計画書
## 「糖尿病注射薬（保険収載）の全種類」と「医療用針の全種類」調査

**作成日：** 2026-04-14  
**用途：** Google NotebookLM への資料投入・調査指示用  
**調査の目的：** 在宅自己注射の互換性検索ツール用データベースを完成させるため、現在のDBに欠落している薬剤・針データを体系的に把握する

---

## ■ 背景と現状の問題

現在のデータベース（c101_medications.csv / compatibility.csv）には以下の不足がある：

- 糖尿病注射薬：約104レコード収録だが、保険収載の全品目を網羅できていない可能性あり
- 針データ：3メーカー（BD・テルモ・ニプロ）のみ。針の規格・長さのバリエーションが不完全
- データ根拠：約65%が「初期データ」（根拠文書なし）
- 新薬対応不足：マンジャロ注（チルゼパチド）、アウィクリ注（週1回型インスリン）が未確認

## ■ 調査テーマ 1：保険収載の糖尿病注射薬 全品目

### 1-1. 調査したいこと

日本で健康保険が適用される糖尿病治療用の注射薬を、以下の分類ごとに全品目リストアップしてほしい。

**収集する情報（各品目について）：**

| 項目 | 内容 |
|------|------|
| 一般名（成分名） | 例：インスリン アスパルト |
| 販売名（商品名） | 例：ノボラピッド注フレックスタッチ |
| 製造販売業者（メーカー） | 例：ノボ ノルディスク ファーマ |
| 剤形・デバイス種別 | プレフィルドペン / カートリッジ / バイアル / 一体型使い捨て |
| デバイス名称 | 例：フレックスタッチ、ソロスター、KwikPen |
| 濃度・用量 | 例：100単位/mL、200単位/mL |

| 在宅自己注射指導管理料 適用有無 | 対象 / 対象外 |

## ■ 調査テーマ 2：医療用針の全種類

### 2-1. 調査したいこと

日本で医療・在宅自己注射に使われる針製品を種類別に全品目リストアップしてほしい。

**収集する情報（各製品について）：**

| 項目 | 内容 |
|------|------|
| 製品名（販売名） | 例：ナノパス34 |
| メーカー | 例：テルモ |
| 針の種類（用途区分） | ペン針 / 注射針（シリンジ用）/ 採血針 |
| ゲージ（太さ） | 例：32G / 34G |
| 長さ | 例：3mm / 4mm / 5mm / 6mm / 8mm |
| 接続規格 | JIS T 3226-2 A型 / ルアースリップ / ルアーロック |
| 対応デバイス | 例：全インスリンペン共通 / 専用品 |
| 本体材質・コーティング | シリコンコーティングあり/なし |
| 入数・パッケージ | 例：100本入 |
| 保険適用区分 | 在宅自己注射 対象 / 対象外 |

- シリンジ用採血針
- 真空採血管用針（ホルダー付き）
- 穿刺針（自己血糖測定用）→ 別カテゴリ

---

### 2-3. 接続規格の整理（重要）

針とデバイスの互換性の核心部分。以下の規格を整理：

| 規格名 | 対象デバイス | 特徴 |
|--------|-------------|------|
| JIS T 3226-2 A型（ISO 11608-2） | 全インスリンペン共通針 | ユニバーサル規格。全メーカーのペン本体に使用可能 |
| ルアースリップ | 注射器（シリンジ）1mL以下 | 針を差し込んで固定。小容量向け |
| ルアーロック | 注射器（シリンジ）2mL以上 | 回転してロック。確実固定 |
| 専用接続 | 一部プレフィルド製剤 | 交換不可（一体型デバイス） |
| ノボ ツイスト | ノボ ノルディスク専用 | ※通常のJIS A型との互換性を要確認 |

---

## ■ 調査後のデータ取り込み手順

NotebookLMで調査した結果を、現在のデータベースに取り込む際の確認項目：

### c101_medications.csv への追加
- 新規品目の行を追加（id連番、category_idの正確な分類）
- device_name 欄の埋め合わせ（空欄が多い現状を解消）
- source 列を「PMDA添付文書」「厚労省別表第九」等の正式名称で記載

### compatibility.csv への追加
- 不足している針製品の行を追加
- 根拠が不明な「初期データ」を公式資料で確認・更新
- ペン型セクション：ノボツイスト、6mm/8mm針等の追加

### 優先度

| 優先度 | タスク | 影響範囲 |
|--------|--------|---------|
| 最高 | 糖尿病薬の全品目確認（別表第九との照合） | 検索ヒット率の改善 |
| 高 | ペン針の全サイズ・全メーカー確認 | 針互換性の精度向上 |
| 中 | 注射針（シリンジ用）の根拠資料確認 | データ信頼性 |
| 低 | 生物学的製剤（プレフィルド）の拡充 | カバレッジ拡大 |
