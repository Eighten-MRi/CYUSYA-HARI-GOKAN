# データ正確性・網羅性の検証計画

## Context

現在の互換性データベースは「初期データ」（65%）と「サンプル」（33%）で構成されており、
根拠が明確でないまま公開状態にある。
ユーザーが在宅自己注射の針選択に使うツールとして、データの正確性と網羅性の確認が必須。
検証対象は主に2軸：①各デバイスに対応する針の情報が正しいか、②糖尿病で使われる注射薬・デバイスが全て網羅されているか。

---

## 情報収集方針

- **別表第九**（特掲診療料の施設基準等）: 在宅自己注射対象の**大分類**確認のみ（製品名レベルの詳細は不足）
- **KEGG** ([https://www.kegg.jp/kegg/](https://www.kegg.jp/kegg/)): 承認済み製品の**製品名レベル**での網羅確認
- **公式HP・添付文書・インタビューフォーム**: 各製品の推奨針の情報源
- **Google NotebookLM**: 上記情報の収集・整理ハブとして使用

## 検証タスク（優先度順）

### タスク1：糖尿病薬の網羅性確認【最優先】

**目的**: インスリン製剤・GLP-1受容体作動薬・GIP/GLP-1デュアルアゴニストが全て揃っているか確認する

**確認方法（2段階）**:

**Step 1 — カテゴリ確認**: 厚労省告示「特掲診療料の施設基準等の別表第九」で在宅自己注射指導管理料の対象薬剤の**大分類（インスリン/GLP-1等）**を確認する

- 参照先: e-Gov 法令検索 / 厚労省告示
- 別表第九は製品名レベルの詳細が不十分なため、大分類の把握にとどめる

**Step 2 — 製品名レベルの網羅確認**: KEGG で各分類の承認済み製品を全件確認し、`data/c101_medications.csv` と照合する

- KEGGの医薬品データベースで「インスリン」「GLP-1」「GIP」等で検索
- 未収載の製品を特定してリストアップ

**具体的に確認が必要な薬剤**:

- **マンジャロ注（チルゼパチド）**: GIP/GLP-1デュアルアゴニスト
- **アウィクリ注（インスリン イコデク）**: 週1回型インスリン、プレフィルドペン
- その他KEGGで発見された未収載製品

**バイアル製剤の扱い**:

- ヒューマリンR/N/3/7、ランタス注バイアル → インスリン専用シリンジとの対応付けが現在未実装
- 対応すべきか判断（インスリン専用1mLシリンジ、U-100、27G×13mm等）

**判断基準**: KEGGの製品一覧に存在する薬剤が c101_medications.csv にあれば「網羅済み」、なければ追加対象

---

### タスク2：ペン針の正確性・網羅性確認【優先度2】

**目的**: 各ペン本体に推奨される針の情報が正確かつ十分に登録されているか確認する

**確認方法**（各ペン本体ごとに公式HP・添付文書・インタビューフォームを参照）:

| ペン本体 | メーカー公式の確認先 |
| ------- | ------------------ |
| ノボペン5 / ノボペン エコーPlus | ノボ ノルディスク ジャパン 公式サイト |
| フレックスタッチ | 同上 |
| ヒューマペン サビオ | 日本イーライリリー 公式サイト |
| ミリオペン | 同上 |
| ソロスター | サノフィ 公式サイト |

**確認項目**:

- 公式が推奨するペン針の種類（ノボツイスト、マイクロファインプラス、ナノパス34、ニプロペンニードル等）
- ゲージ・長さの推奨範囲（3mm / 4mm / 5mm / 6mm / 8mm）
- 現在の4種類（31G×5mm, 32G×4mm, 34G×3mm, 34G×4mm）で実用上十分か

**追加候補として検討**:

- ノボツイスト（ノボノルディスク自社製、32G×4mm）
- マイクロファインプラス 32G×6mm（BD）
- マイクロファインプラス 32G×8mm（BD）
- ナノパス34 34G×5mm / 6mm（テルモ）

**判断基準**: メーカー公式の「使用可能針一覧」に記載あり → `compatibility.csv` に追加

---

### タスク3：注射器+針の正確性確認【優先度3】

**目的**: 「初期データ」65行の根拠を確認し、信頼度を明示する

**確認方法**:

1. テルモ公式サイト「注射針適合シリンジ一覧」と比較
2. ニプロ公式サイト「注射針・シリンジ適合表」と比較
3. 「ルアーロック注射器にルアースリップ針」の組み合わせの注意事項を確認
   - 現在 notes に「固定なし」と記載済み → 表現が適切か確認

**更新アクション**:

- 確認できたものは `source` を「テルモ公式」「ニプロ公式」等に更新
- 確認できなかったものは `compatibility` を「要確認」に変更

---

### タスク4：プレフィルド製剤の網羅性確認【優先度4】

**目的**: 在宅自己注射対象の生物学的製剤で、プレフィルド/オートインジェクター型のものが揃っているか確認

**現在の登録（3件）**:

- ヒュミラペン（アダリムマブ、アッヴィ）
- エンブレルペン（エタネルセプト、ファイザー）
- シムジアオートクリックス（セルトリズマブ、UCB）

**追加候補（c101_medications.csvには登録済みだが compatibility.csv に未登録）**:

- デュピクセント（デュピルマブ、サノフィ）— プレフィルドシリンジ/オートインジェクター
- コセンティクス（セクキヌマブ、ノバルティス）— センソレディペン
- スカイリジ（リサンキズマブ、アッヴィ）— シリンジ注射
- アクテムラ（トシリズマブ、中外製薬）— プレフィルドシリンジ
- エンスプリング（サトラリズマブ）— プレフィルドシリンジ

**判断基準**: c101_medications.csv に登録済みで、かつ在宅での自己注射が可能な製剤 → `compatibility.csv` に追加

## 実施手順

1. **Webリサーチ**（Google NotebookLM で収集・整理）
   - 各タスクの公式情報をKEGG・各メーカーサイト・PMDA添付文書から収集
   - 収集結果を `data/sources.md` に追記（参照URL・確認日を記録）

2. **差分リスト作成**
   - 追加すべき行のCSVデータを一時ファイル（`data/verification_notes.md`）に整理
   - 削除・修正すべき行を特定

3. **CSVファイル更新**
   - `data/compatibility.csv` に追加・修正
   - `data/c101_medications.csv` に不足薬剤を追加
   - `source` 列を「メーカー公式」「PMDA添付文書」等に更新

4. **動作確認**
   - ブラウザで各検索パターンをテスト
   - 変更した薬剤・デバイスが正しく表示されるか確認
