## 参考：把握したいポイント

### 互換性の基本ルール
- **JIS T 3226-2 A型準拠**のペン本体 → BD・テルモ・ニプロ等のペン針がすべて使用可能
- **一体型（交換不可）**のデバイス → オートインジェクター・プレフィルド一体型は針交換不可

### 特に確認したいケース

| ペン本体 | 確認したい点 |
|---------|------------|
| フレックスタッチ（ノボラピッド、トレシーバ等） | JIS A型規格で汎用針が使えるか |
| ノボペン5 / ノボペン エコーPlus | ノボツイスト専用か、汎用でもOKか |
| ヒューマペン サビオ（ヒューマログ等） | JIS A型で汎用針が使えるか |
| ミリオペン（KwikPen） | プレフィルドペン一体型か、カートリッジ式か |
| ソロスター（ランタス、アピドラ等） | 一体型で針交換不要か |
| ジュニアスター | 専用品のみか |
| オゼンピック注専用ペン | 自動注入ペンで針交換不要か |
| トルリシティ アテオス | 一体型で針交換不要か |
